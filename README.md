# Launchpad

An opensource launchpad for macOS!

As you all know, macOS 26 has removed the good old-school launchpad. Therefore, let's make one for it.

# Features

- Fullscreen launchpad UI
- Written in SwiftUI, easy to hack
- Shows your apps
- Launches your apps
